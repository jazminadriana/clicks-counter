# 🌸 Click Counter - React & Glassmorphism

¡Hola! Soy **Jazmín**, una aspirante a **Junior Web Developer** basada en la CDMX. Este es un proyecto interactivo desarrollado con **React** y **Vite** para practicar la gestión de estados y la arquitectura de componentes reutilizables.

[Ver Demo en Vivo] https://jazminadriana.github.io/clicks-counter/

---

## Características (Features)

- **Diseño Glassmorphism:** Estética moderna con transparencias, bordes suaves y una paleta de colores *Soft Pink*.
- **Manejo de Estado Pro:** Implementación de actualizaciones de estado funcionales para garantizar la sincronización.
- **Componentes Reutilizables:** Estructura limpia dividida en componentes lógicos.
- **Desarrollo Rápido:** Configurado con **Vite** para una experiencia de desarrollo ágil.

## Tecnologías Utilizadas

* **React 18** (Functional Components & Hooks)
* **Vite** (Build Tool)
* **CSS3** (Flexbox, Glassmorphism & Custom Transitions)
* **JavaScript (ES6+)**

## Lo que aprendí en este proyecto

Como parte de mi camino en el **Full Stack Open** de la Universidad de Helsinki y **FreeCodeCamp**, en este repositorio reforcé:
1.  El flujo de datos unidireccional (Props).
2.  La importancia de las actualizaciones funcionales en `useState`.
3.  La aplicación de estilos avanzados de UI para mejorar la experiencia de usuario (UX).
